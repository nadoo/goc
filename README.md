# goc
a simple util for building go programs

## Install
Go Get
```bash
go get -u github.com/nadoo/goc
```
## Usage

Change current directory to your package dir, then

- build package
    ```bash
    goc b
    ```

- build package for linux
    ```bash
    goc bl
    ```

- install package to GOPATH
    ```bash
    goc i
    ```
