# goc
a simple compile tool for go

## Install
```bash
go get -u github.com/nadoo/goc
```
## Usage

Change current directory to your package dir, then

- build package
    ```bash
    goc b
    ```

- build package for linux
    ```bash
    goc bl
    ```

- install package to `GOPATH/bin`
    ```bash
    goc i
    ```
